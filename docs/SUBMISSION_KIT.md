## Short Description

Codex Small Business Skills packages 31 practical owner-operator workflows as
Codex skills. It helps AI agents assist with cash-flow snapshots, overdue
invoice follow-up drafts, month-end prep, lead triage, CRM cleanup, customer
support replies, content strategy, campaign planning, hiring packets, and
operating briefs, with approval gates before customer, money, CRM, or contract
actions.

## Long Description

This project ports Anthropic's public Apache-2.0 Small Business plugin skills
into Codex-ready skill folders. The result is a practical SMB workflow pack for
owner-operators, consultants, agencies, and local businesses that want AI agents
to help with the operational work around money, customers, sales, marketing,
hiring, and weekly rhythm.

The port keeps the upstream workflow intent, prefixes each skill for Codex
routing, adds an installer, and documents attribution and modifications. It is
not a financial, tax, legal, accounting, employment, or HR advisor; it is a
workflow-assistance toolkit that keeps risky actions approval-gated.
