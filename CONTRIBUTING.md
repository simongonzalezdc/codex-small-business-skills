# Contributing

Thanks for improving Codex Small Business Skills.

## Ground Rules

- Keep this repository public-safe. Do not add customer data, private business
  context, CRM exports, invoices, payroll records, tax files, tokens, logs, or
  generated business outputs.
- Keep upstream attribution intact.
- Document meaningful modifications from Anthropic's upstream
  `knowledge-work-plugins/small-business` source.
- Preserve approval gates for actions involving customers, money, CRM records,
  contracts, hiring, support, or external publishing.
- Keep legal, tax, financial, accounting, employment, and HR outputs framed as
  workflow assistance, not professional advice.

## Good Contributions

- Codex compatibility improvements.
- Clearer install and troubleshooting docs.
- Tests or validators for skill folder structure.
- Safer connector-fallback language.
- Better examples that use fictional data only.

## Pull Request Checklist

- [ ] No private data or credentials.
- [ ] Apache-2.0 license and attribution preserved.
- [ ] `SKILL.md` frontmatter name matches the directory name.
- [ ] Any risky workflow keeps human approval gates.
- [ ] README or audit docs updated when behavior changes.
